How does the UI relate to how we define interactivity?
This week, we looked more closely at the user interface, specifically how it serves as the system's
communication channel when interaction occurs. When it comes to defining UI and interaction, they are as
follows: A graphical representation of a software or system that allows people to interact with it is referred
to as a user interface (UI). Interactivity refers to how and to what extent the user may engage with the UI and
the system it represents. Interactivity is related to UI since the interface expresses, allows, and embodies
interaction. As users engage with the system to get a source of response (output), the UI not only shows how
users may interact with the system, but it also communicates/provides feedback on the sought result.
